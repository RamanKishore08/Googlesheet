How to Use Formulas:

Go to the Formula tab below the Function Buttons and Type for the following Functions

🔢 Basic Arithmetic Formulas
B1 = A1 + A2 → Adds values of A1 and A2
C1 = A1 - B1 → Subtracts B1 from A1
D1 = A1 * B2 → Multiplies A1 by B2
E1 = A2 / B2 → Divides A2 by B2
F1 = (A1 + B1) * C1 → Uses parentheses for operations


🧮 Mathematical Functions
SUM(A1:A3) → Adds all values from A1 to A3
PRODUCT(A1:A3) → Multiplies all values from A1 to A3
MAX(A1:A3) → Returns the highest value in A1 to A3
MIN(A1:A3) → Returns the lowest value in A1 to A3


🔍 Text Functions
UPPER(A1) → Converts text in A1 to uppercase
LOWER(A1) → Converts text in A1 to lowercase
TRIM(A1) → Removes extra spaces from text in A1
